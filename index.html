Giải thuật bullue sort
  • Bước 1: Gán i = 0
  • Bước 2: j = array.length - 1
  • Bước 3: Nếu array[j] < array[j - 1]
  • Đúng: Thực hiện đổi chỗ array[j] và array[j - 1]
  • j = j - 1
  • Bước 4: Nếu j > i thì quay lại bước 3
Thuật toán sắp xếp nổi bọt bullue sort
Độ phức tạp của thuật toán bullue sort do dùng hai vòng lặp lồng vào nhau nên độ phức tạp thời gian trung bình của thuật toán này là O(n2). Thời gian chạy: ms
Giải thuật đổi chỗ trực tiếp Interchange Sort
  • Bước 1: i = 0;
  • Bước 2: j = i + 1;
  • Bước 3:
  • Khi j < n thì kiểm tra: nếu a[j] < a[i]
    thì hoán vị a[j] và a[i].
  • Gán j = j + 1; rồi thực hiện lại bước 3.
  • Bước 4: i=i+1; nếu i < n-1 thì lặp lại bước 2, ngược lại -> Dừng.

Giải thuật sắp xếp chèn trực tiếp insertion sort
  • 1: Gán i = 1
  • 2: Gán x = A[i] và pos = i – 1
  • 3: Nếu pos >= 0 và A[pos] > x
  • A[pos + 1] = A[pos]
  • pos = pos – 1
  • Quay lại bước 3
  • 4: A[pos + 1] = x
  • 5: Nếu i < n
  • Đúng thì i = i + 1 và quay lại bước 2
  • Sai thì dừng lại
Thuật toán sắp xếp chèn trực tiếp insertion sort
Độ phức tạp của thuật toán chèn trực tiếp insertion sort Độ phức tạp về thời gian: O (n * 2) Thời gian chạy: ms
Giải thuật chọn trực tiếp Selection Sort
  • 1. Gán i = 0
  • 2. j = j + 1 và min = A[i]
  • 3. Nếu j < n
  • Nếu Nếu A[j] < A[min] thì min=j
  • j = j + 1
  • Quay lại bước 3
  • 4. Đổi chỗ A[min] và A[i]
  • 5. Nếu i < n – 1
  • Đúng thì i = i + 1 và quay lại bước 2
  • Sai thì dừng lại

Sắp xếp chèn trực tiếp dựa trên tìm kiếm nhị phân - Binary Insertion Sort
  • 1. gán i = 1
  • 2. gán l = 0, r = i - 1, x = array[i]
  • 3. Lặp nếu l < r
  • m = (l + r)/2 giá trị ở giữa
  • Nếu m > x thì r = m -1 mảng bên trái
  • else r = m + 1 mảng bên phải
  • 4. lặp Nếu array[j] = array[j - 1]
  • array[l] = x;
